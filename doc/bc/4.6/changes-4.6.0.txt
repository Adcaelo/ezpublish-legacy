Changes to BC and behavior in version 4.6.0
===========================================

Change of behavior
------------------

- Template Resource classes must extend eZTemplateFileResource

  eZ Template now uses instance of instead of simply is_object() against resource objects.


- Template SectionIterator classes must extend eZTemplateSectionIterator

  eZ Template now uses instance of instead of simply is_object() against section iterator objects.


- Template Image objects must extend eZImageInterface

  eZ Template now uses instance of instead of simply is_object() against image objects.
